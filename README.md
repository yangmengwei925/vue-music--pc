# toby

> vue-music

## 基于Vue框架开发的webApp音乐播放器0.10版本

demo查看地址: [demo地址:music.itobys.com/](http://music.itobys.com/)

## 项目所用的命令

``` bash
# 拉取项目下来后请先执行以下命令
npm install

# 项目启动以下命令
npm start

# 需要连同server一同启动 需要命令行定位到server目录下执行以下命令（如安装nodemon可利用nodemon替换以下node）
# 该命令可以直接运行项目
node app.js

# 项目打包（打包后请放在server端，用node启动server端绑定到打包后的目录）
npm run build


```
